# stm32f407vet6_display_fsmc_ili9488_ili9341_st7789_xpt2046
Connecting a display with a parallel interface to a popular debug board based on the stm32f407vet6 microcontroller (FSMC with DMA). Connecting an XPT2046 resistive touchscreen controller. JPEG decoder, line-by-line rendering of primitives filled with a solid color, texture, gradient, etc. (CMSIS only)
